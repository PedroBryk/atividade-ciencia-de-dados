// Exercícios

Resolva os desafios

1
exploração inicial Carregando e conhecendo o dataset

Carregue o arquivo vendas_ecommerce.json em um DataFrame e responda:

  • Carregue o JSON em um DataFrame chamado df
  • Exiba as primeiras 5 linhas do DataFrame
  • Mostre os tipos de dados de cada coluna
  • Obtenha um resumo estatístico de todas as colunas numéricas
  • Verifique se há valores nulos em alguma coluna

Após explorar: qual coluna tem o maior valor máximo? E qual a média de avaliacao?

2
nova coluna calculada Calculando o valor final de cada pedido

Crie uma nova coluna chamada valor_final com o valor total pago pelo cliente, descontando o percentual de desconto e multiplicando pela quantidade.

  • Use as colunas quantidade, preco_unitario e desconto_pct
  • Arredonde o resultado para 2 casas decimais
  • Mostre as colunas produto, quantidade, preco_unitario, desconto_pct e valor_final
💡 Dica: você pode operar diretamente entre colunas do DataFrame, como se fossem variáveis numéricas.

Qual o valor_final do pedido de maior desconto percentual?

3
filtragem de dados Selecionando registros específicos

Use filtros booleanos para responder às perguntas abaixo. Cada item deve ser resolvido em uma ou duas linhas de código:

  • 3a. Liste todos os pedidos que foram devolvidos. Mostre: cliente, produto e valor_final.
  • 3b. Filtre os pedidos da categoria "Eletrônicos" com avaliação igual a 5.
  • 3c. Quais pedidos tiveram desconto acima de 15% e valor_final maior que R$ 300?
  • 3d. Encontre o pedido de maior valor_final do dataset.
  • 3e. Quantos pedidos foram feitos por clientes do estado de São Paulo (SP)?

Quantos pedidos foram devolvidos no total? Qual a porcentagem em relação ao total?

4
agregações Agrupando e resumindo os dados

Agrupe os dados para extrair insights de negócio. Para cada item, pense em qual função de agregação faz mais sentido:

  • 4a. Qual o total de receita (soma de valor_final) por categoria? Ordene do maior para o menor.
  • 4b. Qual a avaliação média por método de pagamento?
  • 4c. Qual o número de pedidos por estado?
  • 4d. Qual a média de idade dos clientes por categoria?
  • 4e. Qual categoria tem o maior ticket médio (média de valor_final)?
💡 Dica: pesquise sobre groupby() na documentação do pandas. Você pode encadear métodos como .sum(), .mean(), .count() e .sort_values().

Qual categoria gerou mais receita? Qual gerou menos? Existe alguma surpresa nos resultados?

5
análise temporal Explorando a evolução das vendas por mês

A coluna data está como texto. Você precisará convertê-la para o tipo correto e extrair informações temporais:

  • Converta a coluna data para o tipo datetime
  • Crie uma coluna mes extraindo o número do mês
  • Calcule o total de vendas (valor_final) por mês
  • Calcule o ticket médio por mês (média do valor_final)
  • Em qual mês houve o maior número de pedidos?
💡 Dica: pesquise sobre pd.to_datetime() e como acessar partes de uma data com o acessor .dt.

As vendas cresceram ao longo dos meses? O que os dados sugerem?
